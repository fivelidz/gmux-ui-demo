Overview

What is gmux?

gmux is a gesture-aware AI terminal multiplexer — a unified control center for all your AI coding agents.

Monitor Claude Code, OpenCode, Aider, and Cursor in a single UI. See at a glance who's working, who needs approval, and who's erroring.

Navigate and control everything with hand gestures via MediaPipe or voice commands via browser STT or local Faster-Whisper. No mouse required.

Perfect for multi-monitor setups, standing desks, or hands-on hardware work where you can't reach the keyboard.

Getting Started

Quick Start

Up and running in under two minutes. No install required — runs entirely in the browser.

1
Open the UI
Navigate to http://localhost:5550/v2/index.html in Chrome, Brave, or Chromium.
2
Enable gestures
Press G or click the ✋ Gesture button in the toolbar.
3
Allow camera
Your browser will prompt for camera access — click Allow. The camera stream never leaves your device.
4
Wave your hand
MediaPipe hand tracking loads (~8 MB, cached after first run). Hold your hand in front of the camera.
5
Enable voice
Press V, CapsLock, or use the left-hand ☝ Point gesture to toggle voice listening.
6
Speak
Say "Approve the research agent" — gmux routes the command to the right agent automatically.

Voice Commands

Voice STT — speak to your agents

Two STT backends, both active simultaneously. Browser STT for instant response, Faster-Whisper for accuracy.

🌐 Browser STT Web Speech API
  • Works in Chrome, Brave, Chromium (no install)
  • Streaming — words appear as you speak
  • 👍 Thumbs up gesture to send the draft
  • Transcription shown live in the command bar
"refactor the login function" → sent to selected agent chat
⚡ Faster-Whisper STT local · offline
  • Better accuracy, works in any browser
  • Fully offline — no cloud, no API key
  • Connects automatically on ws://localhost:8770
  • Supports English, French, Spanish, German, Japanese + 90 more

Permission Modes

🔒
Safe
Asks before every tool write or execution. Maximum oversight — nothing runs without explicit approval. Best for sensitive codebases.
⚠️
Yolo Restricted
Auto-approves reads and writes but blocks rm -rf, dangerous shell commands, and system-level writes. Sweet spot for most workflows.
⚡
Yolo Extreme
Fully autonomous — no prompts, no approval gates. Use only in isolated, disposable environments (Docker, VMs). Maximum speed.

Integration

Backend Integration

gmux exposes a full REST+SSE API for external tools, scripts, and phone remotes.

⚙️
OpenCode REST API
Full HTTP+SSE integration. Auto-connects to all running OpenCode instances. Port discovered live via /proc inspection — no config needed.
📡
Live State via HTTP
Monitor exposes /api/state (JSON snapshot) and /api/stream (SSE event stream) on port 8769. Accessible from any device on your network.
📱
Phone PWA
Point your phone browser to http://<your-ip>:5550/mobile/ for a full voice+touch remote. Approve agents, send messages — from your couch.
